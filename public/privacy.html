隱私政策

最後更新日期：2026-04-26

感謝你使用 CHIYIGO（「本站」、「我們」）所提供的服務。本隱私政策說明我們在你瀏覽 chiyigo.com 及其子網域、註冊帳號、使用第三方登入、提交需求工單時，如何收集、使用、保存與保護你的個人資料。使用本服務即代表你同意本政策內容。

1. 我們收集的資料

1.1 你主動提供的資料

  • 帳號註冊：電子信箱、密碼（以 PBKDF2-SHA256 雜湊儲存，不保存明碼）。
  • 第三方登入（OAuth）：當你使用 Google、Discord、LINE、Facebook 等第三方帳號登入時，我們會依平台授權範圍取得「使用者識別碼、姓名／顯示名稱、電子信箱、頭像 URL」。我們不會取得你的密碼或好友清單。
  • 需求工單：你提交的姓名、聯絡方式、需求描述、預算、交期等內容。
  • 2FA（雙因素驗證）：若啟用，會儲存 TOTP secret 與備用碼雜湊。

1.2 自動收集的資料

  • 裝置與連線資訊：IP 位址、User-Agent、裝置 UUID（App 端）、登入時間。
  • Cookie 與本機儲存：用於維持登入狀態（HttpOnly Refresh Token Cookie）、主題偏好、語言偏好、PKCE 流程暫存。
  • 分析數據：透過 Cloudflare Web Analytics 收集匿名瀏覽量與效能指標，不使用 cookie 進行個人追蹤。

2. 我們如何使用資料

  • 建立並維護你的帳號、驗證身份、發送驗證信與密碼重設信。
  • 處理你提交的需求工單，並透過 Telegram 通知本站管理員。
  • 偵測異常登入、防止暴力破解、限流濫用請求（rate limiting）。
  • 跨平台統一身份（Web / App / Unity / Unreal）的單一登入（SSO）流程。
  • 持續改善服務品質與安全性。

3. 第三方服務與資料共享

我們不會販售或出租你的個人資料給任何第三方。在以下情況下，我們可能與第三方共享必要資料：

  • OAuth 平台（Google / Discord / LINE / Facebook）：僅在你主動發起登入時，由你向平台授權；本站只接收平台回傳的識別資訊。
  • Cloudflare：作為基礎架構服務商，承載靜態資源、Pages Functions、D1 資料庫與 Web Analytics。
  • Resend：用於發送系統電子郵件（驗證信、密碼重設、2FA 通知）。
  • Telegram Bot API：用於將需求工單通知本站管理員。
  • 法律要求：若依法律程序、法院命令或政府機關合法要求，我們可能必須揭露相關資料。

4. 資料保存期限

  • 帳號資料保存至你刪除帳號為止。
  • Refresh Token 預設 7–90 天（依平台不同），逾期自動輪換或失效。
  • Email 驗證碼、密碼重設 token、PKCE session 等暫存性資料於 1–60 分鐘內過期，並由每日 Cron 清除。
  • 需求工單於你按下「撤銷」後標記 deleted_at，保留審計追蹤。

5. 你的權利

你擁有以下權利，可隨時透過下方聯絡方式行使：

  • 查詢與取得：要求查看我們持有關於你的個人資料。
  • 更正：要求更正不正確的資料。
  • 刪除：透過儀表板的「刪除帳號」功能（需 Email OTP 二次確認）永久刪除資料。
  • 解綁第三方登入：透過儀表板「帳號綁定」區塊取消任意 OAuth 連結（需保留至少一個有效登入方式）。
  • 撤回同意：你可隨時於各平台後台撤回 OAuth 授權。

6. 資料安全

  • 所有連線強制 HTTPS（TLS 1.2+），由 Cloudflare 提供端到端加密。
  • 密碼使用 PBKDF2-SHA256 雜湊；2FA 備用碼以雜湊形式儲存，並使用常數時間比較防止 timing attack。
  • JWT 採 ES256 簽章，敏感操作（解綁、刪帳）需二次驗證。
  • Admin 操作全程留下稽核日誌（admin_audit_log）。
  • 跨來源請求受 CORS 白名單與 Content-Type 驗證雙重保護。

7. 兒童隱私

本服務未針對 13 歲以下兒童設計。若我們發現未滿 13 歲兒童在未經父母同意下提供個人資料，將立即刪除相關資料。

8. 政策變更

我們可能會不定期更新本政策。若有重大變更，將透過電子郵件或站內公告通知。最新版本永遠可於本頁查閱，並以頁首「最後更新日期」為準。
