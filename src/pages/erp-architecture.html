      // 八層架構速覽

      L1 平台層
      企業平台根節點，所有領域共用治理、合規、Feature Flag 與多租戶 Runtime
      L2~L4 領域層
      16 個獨立 L2 領域、各自 L3 子模組與 L4 細項；領域間以 Event 通訊，不共用 DB
      L5~L7 能力層
      Service / Storage / Runtime 三層具體實作：PostgreSQL、Redis、Kafka、Elasticsearch、Vector DB
      L8 部署層
      Kubernetes、Multi-region Failover、Disaster Recovery、Cloudflare CDN、WAF
